# Ghost Video Extension

Расширение для браузера, которое добавляет забавные призрачные эффекты к видео!

## Возможности
- Случайные глитч-эффекты
- Комментарии призрака
- Неожиданные паузы видео
- Настраиваемая частота эффектов

## Установка
1. Скачайте [последнюю версию](https://github.com/siega52/Ghost-Streaming/tree/main/extension)
2. Распакуйте архив
3. В Chrome откройте `chrome://extensions/`
4. Включите "Режим разработчика"
5. Нажмите "Load unpacked"
6. Выберите папку `extension/`

## Использование
Откройте любой сайт с видео (YouTube, Twitch и т.д.) и наблюдайте за разными эффектами
