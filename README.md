# Ghost Video Extension

Браузерное расширение, которое добавляет призрака к вашим видео

## Функции
- Глитч-эффекты во время просмотра видео
- Случайные комментарии призрака
- Неожиданные паузы и изменение громкости
- Настраиваемая частота появлений

## Установка для разработки
1. Клонируйте репозиторий
2. `npm install`
3. `npm run dev` - для разработки
4. `npm run build` - для сборки
